### ゲームの内容
2つのチームに分かれチームで相談しながらブロックを組み合わせてキャラクターを操作し、最終的なスコアを競うゲーム。
  
### ゲームの流れ
・後攻チームがプログラミング→先攻チームがプログラミング→先攻チームのブロックを実行→後攻チームのブロックを実行。この流れを1ターンとし、8回繰り返す。  
・先攻と後攻は毎ターン交代しながら行われる。  
  
### 役割
・チーム内ではブロックを編集する役割と相手チームのブロックを見る役割がある。ブロックを編集する側のプレイヤーは相手チームをブロックを見ることができず、相手チームのブロックが見えるプレイヤーにはブロックが編集できない。  
・相手チームのブロックを見て、自分チームのブロックを組み立てることで有利にゲームを進めることができる。  
・役割は2ターンごとに交代する。また、ペアがいない場合、一人で2つの役割を負う。  
  
### ブロック
・移動ブロック：1マス移動する。各方向3つずつしか使えない。壁・障害物に向かって進もうとしたら無効なプログラムとして扱われる。  
・回転ブロック：その場でキャラクターが方向転換する。  
・設置ブロック：指定した方向に、爆弾または障害物を設置する。指定した方向が壁・障害物・爆弾・コインのいずれかだったり、爆弾を設置する場合に爆弾の残り個数が0だったら、無効なプログラムとして扱われる。  
・破壊ブロック：指定した方向の障害物を破壊する。指定した方向に障害物がなければ無効なプログラムとして扱われる。  
・回収ブロック：指定した方向の爆弾を回収し、爆弾の残り個数を1増やす。指定した方向に爆弾がなければ無効なプログラムとして扱われる。  
  
### スコア
・キャラクターがコインを取ると100点加算され、爆弾の爆風に当たると50点減算される。  
・壁に向かって進む、オブジェクトがあるマスに障害物を設置する、爆弾のない場所で回収ブロックを使うなど無効なプログラムは実行するたびに5点ずつ減算される。  
・スコアが0を下回ることはない。  
  
### オブジェクト
・オブジェクトにはプレイヤーキャラクターA,B、壁、障害物、コイン、爆弾、罠がある。  
・コインは1ターンに一つずつ生成される。  
・爆弾はターン終了時に爆発し、2マス先まで爆風が伸びる。爆風は壁や障害物に当たるとそれ以上は伸びないが、障害物は破壊される。  
・1ターンに設置できる爆弾は3つまで、相手の爆弾を回収することで一時的に設置できる個数を増やすことができる。(次のターンには持ち越されるが設置すると減る)  
・トラップを踏むと、Lならキャラクターが左回転、Rなら右回転し、踏まれるまでブロックを編集する側のプレイヤーにしか見えない。  
・トラップは毎ターン3つ生成され、2ターンごとに全て破壊され、新たにトラップが生成される。(1ターン目:3つ、2ターン目:6つ、3ターン目:3つ、…)
